# LinkedIn Campaign Setup Checklist

Use this QA checklist before clicking "Launch" on any campaign.
Skipping these steps is the fastest way to waste thousands of dollars.

## 1. Tracking & Audiences
- [ ] Insight Tag is actively firing on the target landing page.
- [ ] Conversion events (if using Landing Pages) are testing as Active.
- [ ] Audience expansion / LinkedIn Audience Network (LAN) is **DISABLED**.
- [ ] Exclusions applied: Competitors, Current Employees, Current Customers.
- [ ] Audience size is appropriate for the goal (> 20k for cold, >1k for retargeting).

## 2. Bidding & Budget
- [ ] "Maximum Delivery" is turned **OFF** for new campaigns (unless strict ABM <20k).
- [ ] Manual CPC is set 10-15% below LinkedIn's high-end recommendation.
- [ ] Daily budget aligns with the 10x target CPA rule.
- [ ] Start date and (if applicable) end date are correct to avoid weekend spend (if testing B2B).

## 3. Creative & Copy
- [ ] Minimum 3, maximum 5 creatives per campaign (to test without starving budget).
- [ ] "Introductory Text" is under 150 characters before the "See More" cutoff.
- [ ] Ad headlines are punchy (under 70 characters).
- [ ] UTM parameters are appended to every single destination URL.
- [ ] Images/Videos pass the mobile-crop test (text is readable on small screens).

## 4. Lead Gen Forms (If Applicable)
- [ ] Form only asks for fields absolutely necessary (Email, First, Last, Company).
- [ ] Privacy Policy URL links to the correct page on your website.
- [ ] "Confirmation Message" contains the actual deliverable link.
- [ ] Zapier/n8n connection is tested via the "Send Test Lead" function.

## 5. Final Sanity Check
- [ ] Campaign naming convention correctly identifies: `[Funnel Stage] - [Objective] - [Audience] - [Date]`.
- [ ] Campaign is assigned to the correct Campaign Group.
- [ ] Billing method is active and has sufficient credit limit.
